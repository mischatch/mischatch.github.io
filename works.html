Flounce

Flounce app is a full-stack web application inspired by Behance. It uses Ruby on Rails on the backend, with a PostgreSQL as it's database and React.js with a Redux framework on the frontend.

Airless

Airless is a platforming game where the aim is to escape collision between blue circle(operated by users mouse) and red tiles for as much as you can.

UnderDOMe

UnderDOMe is a DOM manipulation library inspired by jQuery and writen using JavaScript. It supports some of the most important jQuery features.
